# Rock, Paper, Scissors. 
## This is a small webpage developed as part of a 30 minute introductory tutorial into front-end development. 

### For further learning, clone or download the repository and try personlizing it. Here are some suggestions to get started.
1. Dynamically resize the images. Make the selected image larger and the two others smaller.
2. Restyle the webpage. Change colors, fonts, and positioning. Add more interesting styling for the button.
3. Instead of displaying the results with an alert, make a custom div that pops up with the game result.
4. Add functionality to keep track of and display a game score.
5. Rebuild the site using a tool or resource you want to learn about, like [React](https://react.dev/).
